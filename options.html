Stream Sniper Settings

Appearance

Use dark theme for the extension interface (easier on the eyes in low light)

Notifications

Show a notification when a new stream is detected on a page

Stream Validation

Automatically check if detected streams are accessible (may cause additional network requests)

Video Preview

Automatically load live video previews for all detected streams. Disable this if you have a slow connection or many streams are detected.
Choose preview quality. Lower quality uses less bandwidth and loads faster. Higher quality provides better preview but uses more resources.

Data Management

Automatically save all detected streams to persistent history

Download Settings

Auto-detect uses native downloads for simple streams and yt-dlp for complex ones (if available)
Choose whether to select quality manually or automatically pick the best quality
Downloads will use the closest available quality to this resolution
Display notifications when downloads start, complete, or fail
Automatically remove completed downloads from the Downloads tab after 5 minutes
Customize how downloaded files are named using template variables
Choose a preset or create your own custom template
Available variables:
{title} - Page title
{domain} - Website domain
{quality} - Video quality (1080p, 720p, etc.)
{date} - Current date (YYYY-MM-DD)
{time} - Current time (HH-MM-SS)
{type} - Stream type (m3u8, mpd, etc.)
{timestamp} - Unix timestamp
